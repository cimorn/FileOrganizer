# 更新日志

**中文** | [English](CHANGELOG_EN.md)

本文件记录文件整理的重要变更。

## [26.07.02] - 2026-07-02

### 新增

- 软件名称统一为 `文件整理`，英文说明中使用 `File Organizer`。
- 支持按时间线浏览图片、视频和普通文件。
- 支持自定义批量改名模板，包括 `{yy}`、`{MM}`、`{dd}`、`{index}` 和 `{name}`。
- 支持移动和“改名并移动”的文件夹模板。
- 支持 Excel 导出、导入和表格改名。
- 支持通过 Excel 修改后缀；新后缀为空时保留原后缀。
- 支持按固定间隔批量修改文件创建时间、修改时间和访问时间。
- 支持按大小和 SHA-256 哈希查找重复文件。
- 支持操作历史和撤销改名、移动操作。
- 执行文件操作前会检测目标冲突。
- 为首个开源候选版本生成产品检测报告。

### 调整

- 将 UI 统一调整为蓝色系。
- 将搜索、文件类型、包含子文件夹、页面切换和 Excel 操作移到顶部工具栏。
- 将批量操作按钮和目标文件夹选择放到右侧预览区域。
- Windows 发布包改为 `FileOrganizer-V26.07.02-win-x64.zip`，解压后包含完整可运行文件夹。
- 将项目元信息改为 `file-organizer`、`local.file-organizer` 和 `文件整理`。

### 修复

- 修复打包后主进程路径错误导致的启动失败。
- 修复顶部工具栏和空状态页面的排版问题。
- 清理蓝色主题更新后残留的旧绿色样式。

### 已知说明

- 如果 Windows exe 正在运行，`pnpm run build:desktop` 可能因为 `dist\文件整理-v26.07.02.exe` 被占用而失败。重新打包前请先关闭软件。
- `package.json` 使用 `26.7.2`，因为 npm 包版本需要符合语义化版本规则。公开发布版本仍为 `26.07.02`。
